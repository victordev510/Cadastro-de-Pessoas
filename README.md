Sobre o projeto

O projeto de Cadastro de Usuários foi desenvolvido em Python com o objetivo de criar um sistema simples para armazenar e gerenciar informações de usuários. O aplicativo permite cadastrar dados como nome, idade e e-mail, proporcionando uma experiência prática no desenvolvimento de sistemas organizados e funcionais.

Durante a criação do projeto, foram utilizados conceitos fundamentais da programação, como variáveis, listas, dicionários, estruturas condicionais, laços de repetição e funções, além da interação com o usuário através do terminal.

______________________________________________________________________________________________________________________________________________________________________
Descrição do Projeto ⚙️

O sistema foi desenvolvido para facilitar o registro e a organização de usuários em uma estrutura simples. O programa permite adicionar novos cadastros, visualizar informações registradas e trabalhar com manipulação de dados em Python.

Esse projeto contribuiu para o aprendizado sobre armazenamento de informações, validação de dados e desenvolvimento de menus interativos, fortalecendo a lógica de programação e a organização do código.

Conclusão ✅
